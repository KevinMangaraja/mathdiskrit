Data Understanding - Dataset Iris

1. Informasi Dataset

  • Jumlah Data: 150
  • Jumlah Variabel: 5
  • Tidak ada missing value
  • Distribusi kelas seimbang (50 data per spesies)

2. Scatter Plot Petal Length vs Petal Width

Berdasarkan visualisasi di atas, terlihat bahwa:

  • Iris-setosa terpisah sangat jelas dari dua spesies lainnya.
  • Iris-versicolor dan Iris-virginica masih memiliki beberapa titik berdekatan.
  • Variabel petal_length dan petal_width sangat efektif dalam membedakan spesies.
