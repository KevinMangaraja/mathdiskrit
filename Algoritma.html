1️⃣ Diagram Dua Dimensi (Data Titik)

Diagram yang ditampilkan adalah scatter plot (diagram titik 2D) dengan:

  • Sumbu X : Sepal Length
  • Sumbu Y : Sepal Width

Setiap titik merepresentasikan satu sampel bunga Iris.

Dari grafik dapat diamati:

  • Sebaran data cukup menyebar antara panjang sepal 4.3 – 7.9
  • Lebar sepal berkisar antara 2.0 – 4.4
  • Ada kecenderungan variasi, tetapi tidak membentuk garis lurus sempurna (hubungan tidak sepenuhnya linear kuat)

2️⃣ Perhitungan Statistik Deskriptif

Perhitungan dilakukan untuk seluruh kolom numerik:

  • Sepal Length
  • Sepal Width
  • Petal Length
  • Petal Width

📌 A. Mean (Rata-rata)

Rumus Mean:

x̄ = ∑x / n

Artinya: jumlah seluruh data dibagi jumlah sampel (150 data).

Variabel Mean
Sepal Length 5.8433
Sepal Width 3.0540
Petal Length 3.7587
Petal Width 1.1987

Interpretasi:

  • Rata-rata panjang sepal bunga Iris adalah 5.84 cm
  • Rata-rata lebar sepal adalah 3.05 cm
  • Rata-rata panjang petal adalah 3.76 cm
  • Rata-rata lebar petal adalah 1.20 cm

📌 B. Median (Nilai Tengah)

Rumus Median:

  • Jika n ganjil → data ke-(n+1)/2
  • Jika n genap → rata-rata data ke-n/2 dan (n/2+1)

Karena jumlah data 150 (genap), median diambil dari rata-rata dua nilai tengah setelah data diurutkan.

Variabel Median
Sepal Length 5.80
Sepal Width 3.00
Petal Length 4.35
Petal Width 1.30

Interpretasi:

  • 50% data memiliki panjang sepal ≤ 5.8
  • 50% data memiliki lebar petal ≤ 1.3
  • Median lebih stabil terhadap outlier dibanding mean.

📌 C. Modus (Nilai yang Paling Sering Muncul)

Modus adalah nilai dengan frekuensi kemunculan tertinggi.

Variabel Modus
Sepal Length 5.0
Sepal Width 3.0
Petal Length 1.5
Petal Width 0.2

Interpretasi:

  • Panjang sepal yang paling sering muncul adalah 5.0
  • Lebar sepal yang paling sering muncul adalah 3.0
  • Panjang petal yang paling sering muncul adalah 1.5
  • Lebar petal yang paling sering muncul adalah 0.2
  • Modus menunjukkan nilai dominan dalam dataset.
