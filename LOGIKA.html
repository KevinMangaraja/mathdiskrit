Data Campuran dari Dataset Iris

Dataset Iris asli memiliki 4 atribut numerik dan 1 atribut kategorikal. Untuk membentuk data campuran (numerik, ordinal, kategorikal, dan biner), dilakukan proses transformasi pada beberapa atribut tanpa mengambil data luar.

Penjelasan Pembentukan Tipe Data

  • Numerik: Sepal Length dan Sepal Width digunakan langsung dari data asli karena berbentuk angka kontinu.
  • Ordinal: Petal Length dikategorikan menjadi:
    • < 3 cm → Pendek (1)
    • 3 – 5 cm → Sedang (2)
    • > 5 cm → Panjang (3)
    Karena memiliki tingkatan, maka termasuk data ordinal.
  • Kategorikal: Species (setosa, versicolor, virginica) merupakan label tanpa tingkatan.
  • Biner: Petal Width dikonversi menjadi:
    • ≤ 1.5 cm → 0
    • > 1.5 cm → 1

Kesimpulan

Dataset Iris secara asli hanya memiliki data numerik dan kategorikal. Namun melalui proses transformasi (feature engineering), berhasil dibentuk data campuran yang terdiri dari numerik, ordinal, kategorikal, dan biner. Data campuran ini dapat digunakan untuk analisis lanjutan seperti pengukuran jarak menggunakan metode Gower Distance atau clustering data campuran.

Perhitungan Jarak (Euclidean Distance)

Pada tahap Data Understanding, dilakukan perhitungan jarak antar data menggunakan metode Euclidean Distance. Metode ini digunakan untuk mengukur tingkat kemiripan antar dua sampel data berdasarkan atribut numerik yang dimiliki.

Rumus Euclidean Distance adalah:

d(x,y) = √[(x1 - y1)² + (x2 - y2)² + (x3 - y3)² + (x4 - y4)²]

Dalam perhitungan ini digunakan 4 variabel numerik:

  • sepal_length
  • sepal_width
  • petal_length
  • petal_width

Contoh Perhitungan Manual

Data ke-1: (5.1, 3.5, 1.4, 0.2)
Data ke-2: (4.9, 3.0, 1.4, 0.2)

d = √[(5.1−4.9)² + (3.5−3.0)² + (1.4−1.4)² + (0.2−0.2)²]
d = √[(0.2)² + (0.5)² + 0 + 0]
d = √[0.04 + 0.25]
d = √0.29
d = 0.538516

Analisis

Berdasarkan hasil perhitungan di atas, data ke-4 memiliki nilai jarak paling kecil terhadap data ke-1, yaitu sebesar 0.141421. Hal ini menunjukkan bahwa data ke-4 memiliki karakteristik yang paling mirip dengan data ke-1.

Semakin kecil nilai jarak, maka semakin tinggi tingkat kemiripan antar data. Sebaliknya, semakin besar nilai jarak menunjukkan semakin besar perbedaan karakteristik antar sampel.

Kesimpulan

Dari hasil analisis jarak menggunakan metode Euclidean Distance, dapat disimpulkan bahwa data dalam spesies yang sama cenderung memiliki jarak yang relatif kecil. Perhitungan jarak ini sangat penting dalam algoritma klasifikasi seperti K-Nearest Neighbor (KNN), karena algoritma tersebut menentukan kelas berdasarkan data yang memiliki jarak terdekat.

Dengan demikian, perhitungan jarak membantu memahami pola kemiripan antar data sebelum dilakukan proses pemodelan lebih lanjut.

Perhitungan Jarak Data Campuran (Numerik, Ordinal, Kategorikal, Biner)

1. Pengertian

Pada analisis ini digunakan data campuran yang terdiri dari beberapa tipe variabel, yaitu numerik, ordinal, kategorikal, dan biner. Karena tipe data berbeda, maka metode Euclidean Distance tidak lagi sesuai digunakan secara langsung. Oleh karena itu digunakan metode Gower Distance, yang mampu menghitung jarak untuk data dengan tipe campuran.

2. Pembentukan Data Campuran dari Dataset Iris

Dataset Iris asli memiliki 4 variabel numerik dan 1 kategorikal (species). Untuk keperluan analisis data campuran, dilakukan transformasi sebagai berikut:

  • Numerik: sepal_length
  • Numerik: petal_length
  • Ordinal: ukuran petal (Pendek=1, Sedang=2, Panjang=3)
  • Kategorikal: species
  • Biner: petal_width > 1.0 (Ya=1, Tidak=0)

Variabel ordinal dibuat dengan mengelompokkan panjang petal menjadi 3 kategori:

  • Petal Length < 2.5 → Pendek (1)
  • 2.5 – 5.0 → Sedang (2)
  • > 5.0 → Panjang (3)

3. Rumus Gower Distance

Gower Distance menghitung jarak dengan menyesuaikan setiap tipe variabel, kemudian dirata-ratakan:

D(i,j) = (1/p) × Σ dij

Dimana:

  • Numerik → dihitung berdasarkan selisih yang dinormalisasi
  • Ordinal → dikonversi menjadi skala 0–1
  • Kategorikal → 0 jika sama, 1 jika berbeda
  • Biner → 0 jika sama, 1 jika berbeda

4. Contoh Perhitungan

Misalnya dibandingkan Data ke-1 dan Data ke-2:

Setelah seluruh komponen dihitung dan dirata-ratakan, diperoleh nilai jarak total yang menunjukkan tingkat kemiripan antar data.

5. Analisis

Hasil perhitungan menunjukkan bahwa data yang berasal dari spesies yang sama cenderung memiliki nilai jarak lebih kecil dibandingkan dengan spesies yang berbeda. Variabel kategorikal dan biner membantu mempertegas perbedaan antar kelas, sementara variabel numerik dan ordinal memberikan informasi tingkat variasi ukuran.

Kesimpulan

Pada data dengan tipe campuran (numerik, ordinal, kategorikal, dan biner), metode Gower Distance lebih sesuai dibandingkan Euclidean Distance. Hal ini karena Gower mampu menangani berbagai tipe variabel secara bersamaan tanpa menghilangkan informasi penting dari masing-masing tipe data.

Dengan menggunakan pendekatan ini, analisis kemiripan antar data menjadi lebih akurat dan representatif terhadap kondisi sebenarnya.
