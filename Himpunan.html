Data Understanding - Dataset Iris

1. Pengertian

Data Understanding merupakan tahap awal dalam proses data mining yang berfokus pada proses mengenali, memahami, dan memeriksa kualitas data sebelum dilakukan analisis lanjutan. Pada tahap ini dilakukan pengamatan langsung terhadap isi dataset agar peneliti mengetahui struktur, kondisi, serta kelayakan data.

Tahap ini penting karena kualitas pemahaman data akan sangat mempengaruhi hasil analisis berikutnya.

2. Collect Initial Data

Pada tahap ini dilakukan pengambilan dataset Iris yang digunakan sebagai bahan analisis. Dataset diperoleh dalam bentuk file CSV yang berisi data pengukuran bunga iris. Data kemudian di-import ke dalam aplikasi Orange untuk dilakukan eksplorasi.

Tujuan tahap ini adalah memastikan data berhasil dikumpulkan dan siap digunakan.

3. Describe Data

Berdasarkan tampilan pada Data Table di Orange:

Jumlah Data Jumlah Variabel Missing Value
150 Baris 5 Kolom Tidak Ada

Struktur Dataset:

  • Variabel Numerik:
    • sepal_length (panjang kelopak)
    • sepal_width (lebar kelopak)
    • petal_length (panjang mahkota)
    • petal_width (lebar mahkota)
  • Variabel Kategorikal:
    • species (jenis bunga)

Setiap baris merepresentasikan satu bunga iris beserta ukuran fisiknya.

4. Explore Data

Scatter Plot digunakan untuk melihat penyebaran data antar variabel. Dari grafik terlihat bahwa data membentuk beberapa kelompok berdasarkan spesies, meskipun sebagian masih saling berdekatan.

Correlations digunakan untuk mengetahui hubungan antar variabel numerik. Hasil menunjukkan bahwa terdapat variabel yang memiliki hubungan kuat dan ada yang lemah.

Box Plot digunakan untuk melihat distribusi data. Secara umum data tersebar dalam rentang yang wajar tanpa outlier ekstrem.

Rank digunakan untuk mengetahui tingkat kepentingan variabel dalam membedakan spesies. Terlihat bahwa tidak semua variabel memiliki pengaruh yang sama.

Pivot Table menunjukkan bahwa jumlah data tiap spesies seimbang, masing-masing sebanyak 50 data.

5. Verify Data Quality

  • Tidak ada missing value
  • Tidak ada duplikasi mencolok
  • Format data konsisten
  • Dataset siap dianalisis

Dengan demikian dataset Iris memiliki kualitas yang baik untuk proses eksplorasi menggunakan Orange.

Rank (Pemeringkatan Atribut)

Pada bagian Rank, dilakukan proses pemeringkatan atribut untuk melihat seberapa besar kontribusi masing-masing variabel dalam membedakan spesies bunga iris.

Berdasarkan hasil yang diperoleh, urutan variabel dari yang paling berpengaruh adalah:

Peringkat Variabel Tingkat Pengaruh
1 petal_length Sangat Tinggi
2 petal_width Tinggi
3 sepal_length Sedang
4 sepal_width Rendah

Variabel petal_length dan petal_width menempati peringkat tertinggi, yang menunjukkan bahwa kedua atribut tersebut memiliki peranan paling signifikan dalam proses klasifikasi jenis bunga iris.

Sebaliknya, sepal_width berada pada peringkat terendah sehingga pengaruhnya terhadap pembedaan spesies relatif kecil.

Dengan demikian, dapat disimpulkan bahwa karakteristik pada mahkota bunga (petal) lebih dominan dibandingkan kelopak bunga (sepal) dalam menentukan perbedaan antar spesies iris.

Eksplorasi Dataset Iris Menggunakan Orange

Pada tahap eksplorasi menggunakan aplikasi Orange, dataset Iris digunakan sebagai objek analisis. Berdasarkan tampilan pada Data Table, dataset ini terdiri dari 150 baris data dan 5 atribut. Seluruh data terisi lengkap tanpa adanya missing value sehingga data siap digunakan untuk proses analisis lebih lanjut.

Dataset Iris memiliki empat atribut numerik, yaitu:

  • sepal_length (panjang kelopak)
  • sepal_width (lebar kelopak)
  • petal_length (panjang mahkota)
  • petal_width (lebar mahkota)

Selain itu terdapat satu atribut kategorikal yaitu species yang menunjukkan jenis bunga iris: Iris-setosa, Iris-versicolor, dan Iris-virginica.

Setiap baris data merepresentasikan satu sampel bunga beserta ukuran bagian-bagiannya dan label spesiesnya. Dataset ini sering digunakan dalam pembelajaran machine learning, khususnya untuk kasus klasifikasi, karena strukturnya rapi, mudah dipahami, dan tidak mengandung data kosong.

Visualisasi Scatter Plot

Pada tahap visualisasi data menggunakan fitur Scatter Plot di Orange, dilakukan perbandingan antara dua variabel, yaitu petal_length (sumbu X) dan petal_width (sumbu Y).

Setiap titik pada grafik merepresentasikan satu sampel bunga iris. Perbedaan warna digunakan untuk membedakan kategori spesies.

Berdasarkan visualisasi, kelompok Iris-setosa terpisah sangat jelas dari dua spesies lainnya. Hal ini menunjukkan bahwa ukuran mahkota bunga pada Iris-setosa berbeda cukup signifikan.

Sementara itu, Iris-versicolor dan Iris-virginica masih memiliki beberapa titik yang berdekatan, namun pemisahannya lebih jelas dibandingkan jika menggunakan variabel sepal.

Visualisasi ini menunjukkan bahwa variabel pada bagian petal lebih efektif dalam membantu membedakan spesies bunga iris.

Analisis Correlations

Pada tahap analisis menggunakan fitur Correlations di Orange, dihitung hubungan antar variabel numerik menggunakan metode Pearson correlation.

Nilai korelasi berada pada rentang -1 sampai +1:

  • Nilai mendekati +1 menunjukkan hubungan positif yang kuat.
  • Nilai mendekati -1 menunjukkan hubungan negatif.
  • Nilai mendekati 0 menunjukkan hubungan yang lemah.

Berdasarkan hasil analisis:

  • petal_length dan petal_width memiliki korelasi sangat tinggi (~ +0.963)
  • petal_length dan sepal_length memiliki korelasi kuat (~ +0.872)
  • petal_width dan sepal_length memiliki korelasi cukup kuat (~ +0.818)
  • sepal_length dan sepal_width memiliki korelasi sangat lemah (~ -0.109)

Dapat disimpulkan bahwa variabel pada bagian petal memiliki keterkaitan yang lebih kuat dibandingkan variabel sepal, sehingga lebih informatif dalam membedakan spesies iris.

Box Plot

Pada tampilan Box Plot, ditunjukkan penyebaran data untuk variabel petal_length.

Grafik menampilkan nilai minimum, maksimum, median, dan rata-rata. Nilai rata-rata panjang petal berada di sekitar 3.76, dengan sebaran data yang cukup lebar.

Box Plot membantu memahami distribusi data, tingkat variasi, serta mendeteksi kemungkinan outlier. Berdasarkan grafik, panjang mahkota memiliki variasi cukup besar namun masih dalam rentang yang wajar.

Column Statistics

Pada bagian Column Statistics, ditampilkan ringkasan statistik dari masing-masing variabel, meliputi:

  • Mean (rata-rata)
  • Median
  • Mode
  • Minimum dan maksimum
  • Dispersion (tingkat penyebaran)
  • Missing value

Seluruh data berada dalam kondisi lengkap tanpa missing value (0%). Variabel pada bagian petal menunjukkan variasi yang lebih besar dibandingkan variabel sepal.

Informasi ini membantu memberikan gambaran awal kondisi dataset sebelum dilakukan analisis lanjutan.
